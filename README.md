# morrisjs-rails

asset gem for morris.js

## Installation

  bundler
  
## Usage

Add the following directive to your Javascript manifest file (application.js):

    //= require morris

Add the following directive to your Stylesheet (application.css):

    *= require morris

## Versioning

morrisjs-rails 0.2.4 == morris.js 0.2.4
